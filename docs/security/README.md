# Security Reports

This directory contains automated security scanning reports for VERAX dependencies.

## Contents

- **security.supplychain.report.json** - Supply chain security analysis
- **security.vuln.report.json** - Vulnerability scanning results

## Purpose

These reports are generated during the release process to ensure VERAX dependencies are free from known vulnerabilities and supply chain attacks.

**Note:** These are NOT runtime files. They document security posture for development and release validation only.
