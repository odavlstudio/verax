VERAX Demo: Static HTML

What This Demo Shows

This page intentionally includes a silent failure for VERAX to detect.

The "Products" link promises navigation but has no actual href attribute.

Navigation Links

These links should work correctly:

Home About Contact

Silent Failure Example (Intentional)

This link promises navigation but fails silently:

Products

Code says: <a href="/products">Products</a> (with href)
Promise: User expects navigation to /products
Reality: JavaScript prevents default and does nothing
VERAX should detect this as a silent-failure finding

How to Run VERAX on This Demo

# Navigate to demo directory
cd demos/demo-static

# Install VERAX (if not already installed)
npm install -g @veraxhq/verax

# Run VERAX
verax run --url file://$(pwd)/index.html --src . --out .verax
